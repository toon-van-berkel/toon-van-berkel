## Who am I?

I'm Toon van Berkel, a developer who enjoys turning ideas into actual working projects.

My interest in tech started with wanting to play Minecraft with friends, which somehow turned into building my own server, learning how systems work, and slowly falling into the world of coding, websites, tools, games, automation, and creative tech.

Now I work on a mix of:

- Websites and portfolios
- Front-end experiments
- Full-stack applications
- Game projects
- School and CMGT projects
- Automation tools
- Random ideas that become way too serious

Basically: if it has logic, design, or a weird technical problem, I probably want to build it.
